Detailed Explanation of PCA

Principal Component Analysis (PCA) is a rigorous mathematical procedure that transforms a number of possibly correlated variables into a smaller number of uncorrelated variables called Principal Components. It is the ultimate tool for "Dimensionality Reduction" in hydrogeochemistry, designed to strip away statistical noise and reveal the hidden, underlying natural or anthropogenic processes controlling groundwater quality.

The Geometric Analogy

Imagine shining a flashlight on a 3D cloud of data points to cast a 2D shadow on a wall. Depending on where you hold the light, the shadow might look like a narrow line, or a wide, scattered circle. PCA is the mathematical flashlight. It automatically finds the perfect orthogonal angles (eigenvectors) to shine the light so the resulting "shadow" (your 2D biplot) retains the absolute maximum amount of original variance.

PCA by Covariance Method and Correlation Method

Principal Component Analysis can be performed mainly using two approaches. Both methods follow the same PCA principle, but the data treatment differs. The background engine of this dashboard automatically adapts to whichever method you select.

1. PCA Using Covariance Matrix

Basic Idea & Mathematical Basis

  • Original values are used directly, but the data is mean-centered.
  • Variance and covariance are preserved in their original physical units.
  • Rule: Gives massive mathematical importance to variables having larger variance.

Covariance Formula:

Cov(X, Y) = Σ (Xi - X̄)(Yi - Ȳ) / (n - 1)

When to Use & Pros/Cons

Use covariance PCA when: ✅ All variables have similar units, ✅ Scale differences are small, and ✅ Original variance is important.

Advantages

Retains original variance. Good for similar units. Easier physical interpretation since magnitude matters.

Limitations

Large scale variables completely dominate (e.g., EC vs pH). Different units create massive bias.

Computation Steps in Dashboard (Covariance)

Step 1: Prepare Data Matrix
Step 2: Mean Centering (Subtract X̄)
Step 3: Build Covariance Matrix
Step 4: Solve Ax = λx (Eigenvalues)

2. PCA Using Correlation Matrix

Basic Idea & Standardization

  • Data is standardized first (converted to equal scale).
  • Mean becomes 0, Standard Deviation becomes 1.
  • All variables contribute equally. The most commonly used method in groundwater studies.

Standardization:

Z = (X - μ) / σ

Correlation (r):

r = Cov(X,Y) / (σXσY)

When to Use & Pros/Cons

Use correlation PCA when: ✅ Variables have different units/scales, ✅ Relative relationships are important.

Advantages

Removes unit bias. Best for mixed parameters (ideal for environmental chemistry).

Limitations

Original magnitude is lost. Small-variance variables may gain artificial noise importance.

Groundwater Real-Life Example

Suppose your groundwater parameters are: pH (6–8), EC (500–5000), and As (0.001–0.05).

If using Covariance PCA:

EC will completely dominate the analysis because its values are numerically very large (thousands), silencing the critical variation in Arsenic and pH.

If using Correlation PCA:

All variables are standardized. Now pH, EC, and As all contribute fairly to identifying hidden contamination sources.

Conclusion: Correlation PCA is preferred!

Retention Criteria: How Many Components to Keep?

1. Kaiser & Jolliffe Rules

Standard Kaiser Rule (For Correlation):

Keep components with an Eigenvalue > 1.0.
Example: If you analyze 12 parameters, the total variance is 12. A component with an eigenvalue of 2.5 explains the equivalent variance of 2.5 original variables, so it is retained. A component with an eigenvalue of 0.8 is discarded because it explains less information than a single raw variable.

Jolliffe Rule (For Covariance):

Keep components with an Eigenvalue > Average Eigenvalue. Because covariance eigenvalues can scale into the thousands, the cutoff dynamically adapts to the mean dataset variance.

2. Cumulative Variance

  • Add up the percentage of variance explained by PC1, PC2, PC3, etc.
  • Target: For hydrogeochemistry, aim for a cumulative total of 70% to 85%.

Visualizing PCA: 2D & 3D Interpretations

Graphical representations translate complex mathematical matrices into actionable insights. Here is a breakdown of the interactive charts generated in the Analytics Tab and exactly how to read them.

1. Scree Plot

A line graph displaying the eigenvalues (variance explained) for each principal component in descending order.

How to Interpret:

Look for the "elbow" or sudden flattening of the curve. The steep slope represents the components containing the "signal" (actual hydrochemical processes), while the flat tail represents statistical "noise".

2. 2D Score Plot

A scatter plot representing the physical samples (locations/villages) transformed into the new PCA coordinate system.

How to Interpret:

Points clustered close together share highly similar water quality profiles. An isolated point far from the center is a geochemical outlier (e.g., a highly polluted well). The spread along the X-axis (PC1) explains the most massive variance in the dataset.

3. 2D Loading Plot

A scatter plot displaying the original variables (parameters like pH, EC, As) and how heavily they contribute to each PC.

How to Interpret:

Parameters clustered together (e.g., Na and Cl) are highly positively correlated and originate from the same source. Parameters on exactly opposite sides are negatively correlated. Parameters far from the center (0,0) are the dominant drivers of the system.

4. Unified Biplot

The holy grail of PCA. It overlays the Score Plot (Samples as points) directly on top of the Loading Plot (Variables as lines/vectors).

How to Interpret:

If a specific sample (point) lies far out in the exact same direction as the Fluoride vector line, that specific sample has severely high Fluoride concentrations. It visually connects specific locations to specific geochemical anomalies instantly.

5. Interactive 3D Spatial Map (PC1 vs PC2 vs PC3)

While a 2D plot relies purely on the X and Y axes, a 3D Plot introduces the Z-axis (PC3). This is absolutely critical in highly complex hydrogeochemical systems where 2 components are not enough to explain the data.

When to use it: If PC1 + PC2 only capture a cumulative variance of < 65%, a standard 2D plot hides critical information. The 3D plot visualizes the first three dimensions simultaneously.

Real-World 3D Example:

Suppose PC1 captures natural salinity (EC, TDS) and PC2 captures agricultural runoff (NO3). A 2D plot shows all locations mapped by these two factors.

However, there is a localized Arsenic anomaly in 3 specific villages. Because Arsenic is independent of salinity and farming, it forms PC3. Rotating the 3D map will suddenly reveal these 3 villages floating distinctly higher on the Z-axis, a critical insight completely invisible on a 2D screen!

Common Hydrochemical Signatures (Cheat Sheet)

EC + F + U + Na

  • Prolonged Rock-Water Interaction
  • Deep aquifer / high residence time
  • Geogenic mineral dissolution

NO3 + K + Cl

  • Anthropogenic surface pollution
  • Agricultural fertilizer leaching
  • Domestic wastewater infiltration

Ca + Mg + Alkalinity

  • Carbonate weathering (Limestone)
  • Temporary hardness dynamics
  • Recent recharge zones

Fe + Mn + As

  • Reducing/anoxic aquifer conditions
  • Redox-controlled mobilization
  • Stagnant groundwater pockets
