About the Groundwater Compliance Engine

A comprehensive tool for assessing groundwater quality against BIS IS 10500 standards.

Core Functionality

This engine automates the evaluation of hydrochemical data. It intelligently maps raw Excel data to standard BIS parameters, categorizing samples into Acceptable, Permissible, and Above Limit tiers. It generates visual compliance distributions and comparative exceedance charts dynamically.

Reporting & Export

  • Multi-tier spatial analysis (State, District, Block level aggregation).
  • Excel Report Generation: One click exports comprehensive compliance sub-sheets for every parameter.
  • Exceedance Tracking: Auto-generates detailed logs of locations exceeding limits, including multi-parameter contamination cross-analysis.

Data Formatting Guidelines & Discrepancies

Follow these rules to ensure the engine processes your data accurately and avoids false results.

1. No Non-Numeric Strings in Data

The tool expects numerical values for chemical parameters. Do not use terms like "ND" (Non-Detect), "BDL" (Below Detectable), "Traces", or values with operators like "< 0.05". These will cause the sample to evaluate as NaN (Not a Number) and be excluded entirely, potentially skewing your percentages. Fix: Replace these with 0 or the exact detection limit numerical value.

2. Single Header Row Only

The uploaded Excel file must have exactly ONE header row located at Row 1. Do not use merged cells, double headers, or leave blank rows before the header. The system maps parameter columns dynamically based on this first row.

3. Remove Units from Cells

Do not include units inside the data cells (e.g., writing "15 mg/L" instead of just 15). The engine already knows the standard BIS units and will fail to parse cells containing text.

4. Consistent Location Spelling

When doing District or Block level analysis, exact string matching is used. "New Delhi" and "new delhi " (with a trailing space) will be treated as completely different locations. Please trim and standardize your location text before uploading.
