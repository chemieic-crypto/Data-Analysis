Tool Methodology & Overview

1. Accepted Data Templates

This analyzer dynamically processes two distinct data architectures. For Hypothesis Testing to work, data must be pairable by Well ID or Location across two periods.

  • Standard Format (Long): Data is organized by rows. Each row represents a location at a specific time (e.g., Pre-monsoon 2022). Requires Year and Season columns to group and compare periods.
  • Wide Format: Data is organized by columns. A single row contains all periods for a location. Detects column pairs like ECPRM and ECPOM.
Example: Standard Format (CSV/Excel)
Well_IDYearSeasonTDSNO3
W0012023Pre-Monsoon55042
W0012023Post-Monsoon48038
W0022023Pre-Monsoon120055

2. Core Capabilities

  • Trend Analysis: Automatically computes % change between Base and Compare periods. Classifies locations as Improved (>20% decrease), Deteriorated (>20% increase), or No Significant Change.
  • Permissible Limit Flagging: Cross-references concentrations against WHO/BIS limits (e.g., TDS > 2000, F > 1.5) to highlight critical deterioration.
  • Statistical Hypothesis Testing: Runs Paired T-Tests and Wilcoxon Signed-Rank tests on matched wells to mathematically prove if the seasonal shift is statistically significant.
  • HD IDW Geospatial Interpolation: Uses Inverse Distance Weighting to create continuous, glossy heatmaps constrained perfectly to shapefile boundaries.

3. Hypothesis Testing Explained

To confidently claim that the monsoon significantly impacted groundwater quality, we must prove the change isn't due to random chance. We use paired statistical tests because we are measuring the same wells twice (Base vs Compare).

Student's Paired T-Test (Parametric)

Best for normally distributed data. It measures the mean difference relative to the standard error.

t = d̄ / ( s_d / √n )
  • d̄ = Mean of the differences (Base - Compare)
  • s_d = Sample standard deviation of differences
  • n = Number of valid paired locations
Wilcoxon Signed-Rank (Non-Parametric)

Best for real-world environmental data (resists outliers). Ranks absolute differences to find significance.

W = Σ R(+)
  • R(+) = Sum of ranks for positive differences
  • Ranks are assigned to absolute non-zero differences.
  • Zeros are excluded from n.
Interpreting P-Values & Significance:
• If P-Value < 0.05 (at 95% Confidence), it outputs YES or YES depending on the direction. There is less than a 5% probability of observing a difference this large assuming the monsoon actually had no effect. The impact is significant.
• If P-Value > 0.05, it outputs NO. The change is considered statistically insignificant noise.

4. Analytical Tabs Explained

General Summary Aggregates locations (State/Dist/Block) showing the Count (N) and Percentage (%) of improving vs deteriorating wells based on the 20% thresholds.
Permissible Limits Isolates critical locations. Shows wells that were already beyond limits and deteriorated further.
Contamination Report Evaluates all parameters side-by-side to show newly contaminated and remediated samples.
Hypothesis Testing Validates statistical significance of the shift between periods using Paired T-Test and Wilcoxon tests.
Data Availability Validates data completeness. Shows exactly how many records had valid numbers for both base and compare periods.
Detailed Matrix A complete row-by-row breakdown showing the raw Base value, Compare value, precise % Change, and assigned Status.
Advanced Map High-Definition IDW mapping restricted precisely to uploaded administrative shapefile boundaries.
